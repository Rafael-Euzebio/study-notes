# JSON
JSON (JavaScript Object Notation) is a format to store and exchange information in a human readable way

It was derived from javascript but nowadays is used by most modern programming languages natively

## Structure
JSON uses `key` `value` pairs and arrays to store information

  "is_alive": true,

    "state": "NY",

  },
